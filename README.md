# veirox-scoop

Scoop bucket for the [Veirox CLI](https://veirox.com/cli.html).

## Install

```powershell
scoop bucket add veirox https://github.com/veirox-cloud/veirox-scoop
scoop install veirox
```

## What's here

This repo contains a single Scoop manifest for `veirox`. The manifest
is **auto-updated by GoReleaser** on every CLI release tag — see
[`veirox-cli`](https://github.com/veirox-cloud/veirox-cli) for the
binary source.

## Why this repo is public

Scoop buckets are typically public so `scoop bucket add` can clone them
without authentication. The binary source itself stays in a private
repo.
